# Student Record Management System

## Overview
A Java-based console application to manage student records using Object-Oriented Programming concepts.

## Features
- Add Student
- View Students
- Delete Student
- Menu Driven System
- Console Based Interface

## Technologies Used
- Java
- OOP Concepts
- ArrayList
- Scanner Class

## Project Structure

- StudentManagementSystem.java

- Classes and Objects
- Constructors
- ArrayList
- Loops
- Conditional Statements
- Methods
- CRUD Operations
